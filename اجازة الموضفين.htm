لإنشاء نظام لإجازات الموظفين باستخدام Google Spreadsheet و Google Apps Script، يمكننا إعداد جدول بيانات لتسجيل طلبات الإجازات ومن ثم استخدام Google Apps Script لتطبيق القواعد والإجراءات المطلوبة.

 

### الخطوات الأساسية:

 

#### 1. **إعداد جدول البيانات (Google Spreadsheet):**

 

قم بإنشاء جدول بيانات جديد في Google Sheets يحتوي على الأعمدة التالية:

 

- **الاسم** (اسم الموظف)

- **تاريخ البداية** (تاريخ بداية الإجازة)

- **تاريخ النهاية** (تاريخ نهاية الإجازة)

- **نوع الإجازة** (مثل: سنوية، مرضية، إلخ)

- **الحالة** (موافقة/مرفوضة)

- **ملاحظات** (أي ملاحظات إضافية)

 

مثال على التنسيق:

| الاسم       | تاريخ البداية | تاريخ النهاية | نوع الإجازة | الحالة   | ملاحظات |

|-------------|---------------|---------------|-------------|----------|----------|

| أحمد علي    | 01/12/2024    | 05/12/2024    | سنوية       | مرفوضة   |          |

| مريم حسين   | 10/12/2024    | 15/12/2024    | مرضية       | موافقة   |          |

 

#### 2. **إضافة Google Apps Script:**

 

انتقل إلى "الأدوات" في Google Sheets واضغط على "محرر البرمجة" (Script Editor) لفتح محرر السكربتات.

 

#### 3. **كتابة سكربت للموافقة على الإجازات أو رفضها:**

 

استخدم الكود التالي لإدارة طلبات الإجازة:

 

```javascript

function approveLeaveRequests() {

  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();

  var range = sheet.getDataRange();

  var values = range.getValues();

 

  // الحصول على التاريخ الحالي

  var currentDate = new Date();

 

  // التكرار عبر جميع الصفوف

  for (var i = 1; i < values.length; i++) {

    var startDate = new Date(values[i][1]); // تاريخ البداية

    var endDate = new Date(values[i][2]);   // تاريخ النهاية

    var status = values[i][4];              // الحالة

   

    // إذا كانت الإجازة في المستقبل، قم بالموافقة عليها

    if (currentDate <= startDate && status != "موافقة") {

      sheet.getRange(i + 1, 5).setValue("موافقة");  // تحديث الحالة إلى "موافقة"

    }

    // إذا كانت الإجازة في الماضي أو تم رفضها، يتم تعيين الحالة إلى "مرفوضة"

    else if (currentDate > endDate && status != "موافقة") {

      sheet.getRange(i + 1, 5).setValue("مرفوضة");

    }

  }

}

```

 

#### 4. **إعداد الوظائف التلقائية:**

 

- **التشغيل التلقائي للسكربت:** يمكنك جعل السكربت يعمل تلقائيًا كل يوم عن طريق إضافة مشغل "وقت" (Time-driven Trigger):

  1. اذهب إلى "المحرر البرمجي" (Script Editor).

  2. اختر "المحرر" (Editor) ثم "المشغلات" (Triggers).

  3. اضغط على "إضافة مشغل" (Add Trigger).

  4. اختر الوظيفة `approveLeaveRequests` وحدد "التشغيل يوميًا" (Time-driven) أو حسب التوقيت الذي تفضله.

 

#### 5. **إعداد نموذج (اختياري):**

 

يمكنك أيضًا إعداد نموذج Google Forms لجمع طلبات الإجازة من الموظفين، ثم ربطه بـ Google Sheets:

 

1. أنشئ نموذج Google Forms يتضمن حقول مثل الاسم، تاريخ البداية، تاريخ النهاية، ونوع الإجازة.

2. في إعدادات النموذج، قم بتحديد إرسال الردود إلى Google Spreadsheet.

 

#### 6. **اختبار النظام:**

 

- جرب إرسال بعض البيانات يدويًا عبر جدول البيانات أو النموذج.

- تأكد من أن السكربت يعمل بشكل صحيح ويحدث حالة الإجازات بناءً على التاريخ الحالي.

 

### تخصيصات إضافية:

- يمكنك إضافة إشعارات بريد إلكتروني للموافقة أو الرفض.

- يمكنك إضافة ميزات أخرى مثل تحديد عدد الإجازات المتبقية للموظف أو توفير إشعارات تلقائية.

 

### مثال على إضافة إشعار بالبريد الإلكتروني عند الموافقة:

 

```javascript

function sendApprovalEmail(name, startDate, endDate) {

  var email = "example@example.com"; // البريد الإلكتروني المسؤول

  var subject = "إجازة معتمدة للموظف: " + name;

  var message = "تمت الموافقة على إجازتك من " + startDate + " إلى " + endDate;

 

  MailApp.sendEmail(email, subject, message);

}

```

 

يمكنك استدعاء هذه الدالة بعد أن يتم الموافقة على الإجازة في السكربت.

 

باتباع هذه الخطوات، سيكون لديك نظام فعال لإدارة إجازات الموظفين باستخدام Google Sheets و Google Apps Script.
